# PhishGuard Server

Серверная часть системы обнаружения фишинговых ссылок.

## Назначение
Сервер обрабатывает запросы от браузерного расширения PhishGuard, проверяет ссылки через VirusTotal API и отправляет уведомления пользователям через VK Bot и администраторам через Telegram.

## Взаимодействие с расширением
Для интеграции с браузерным расширением используйте:
- Endpoint: `POST /api/check-result`
- Аутентификация: HMAC-SHA256
- Формат данных: JSON

## Установка
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Создайте `.env` файл на основе `.env.example`
4. Запустите сервер: `python server.py`

## API Endpoints
- `POST /api/check-result` - прием результатов проверки
- `POST /api/report-link` - отчет о ссылках
- `POST /vk-callback` - обработка VK Callback API
- `GET /health` - проверка работоспособности
- `POST /api/check-url` - ручная проверка URL

## Примечание
Браузерное расширение является отдельным компонентом системы и не входит в данный репозиторий.
